# 🛒 Shopping Cart Web Application

A simple and interactive **Shopping Cart Web App** built using HTML, CSS, and JavaScript. This project allows users to select products, adjust quantities, and view the total price dynamically.

---

## 🚀 Features

* 🛍️ Multiple products display
* ➕ Increase / ➖ decrease quantity
* 🛒 Add items to cart
* 💰 Auto total price calculation
* 📦 Dynamic cart update
* 🎨 Simple and clean UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 💡 How It Works

* Each product has:

  * Price
  * Quantity selector
  * Add to Cart button

* User actions:

  * Increase/decrease quantity
  * Click **Add to Cart**

* JavaScript:

  * Reads product price
  * Multiplies by quantity
  * Updates cart list
  * Calculates total amount

---

## ⚠️ Note

* Prices like `50,000` and `30,000` may cause issues in JavaScript calculations.
  👉 Use numeric format like:

```id="n8k2lm"
50000
30000
```
